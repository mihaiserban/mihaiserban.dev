This is the source code for [mihaiserban.dev](https://mihaiserban.dev).

### Requirements for local development

- [Node](https://nodejs.org/en/) (v18+ recommended)
- [Gatsby-cli](https://www.gatsbyjs.org/docs/gatsby-cli/)

### Installing

1. `git clone https://github.com/mihaiserban/mihaiserban.dev.git`
2. `cd mihaiserban.dev && npm install && npm run dev`
3. Site should now be running on http://localhost:8000

### Stack

- Site Generator
  - [GatsbyJS](https://www.gatsbyjs.org/)
- Content
  - Static Markdown & JSON files in Git
- Styling
  - [Tailwind CSS](https://tailwindcss.com/) + [SCSS](https://sass-lang.com/)
- Hosting
  - [Netlify](https://www.netlify.com/)
